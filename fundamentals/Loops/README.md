# Loops
Loops are control structures that execute code blocks repeatedly until a condition is no longer satisfied or until all elements of a collection have been processed. There are two main types: `while` and `for`.
## Loop types
### While
The `while` loop repeats while a condition evaluates to `True`, commonly used when you don't know how many times the loop will execute.

### For
The `for` loop executes code blocks a certain number of times or until all elements of a specific collection have been processed, commonly used when we know the data set or how many times the loop will execute.
> 💡 **Note:** In Python, the `for` loop is commonly used to iterate over collections. In languages such as Java, C and C++, the `for` loop can also be used as a counter.

### Infinite loops
An **infinite loop** happens when a loop condition never evaluates to `False`. It is commonly caused by logical errors, but it is sometimes used intentionally, such as in servers or embedded systems.

### Nested loops
A **nested loop** happens when one loop is inside another. This kind of loop is often used to iterate two-dimensional data structures, such as matrices.

### Control keywords
* **Break:** Immediately exits the loop.
* **Continue:** Skips the current iteration and continues with the next one.

> 💡 **Note:** Python also has the `pass` statement, which defines an empty code block. It is often used as a placeholder when no action needs to be executed.
